## 📝 Sobre o Projeto

O **Indaia Fighter** nasceu como um desafio acadêmico para a disciplina de *Projeto Integrado de Jogos Digitais*. A ideia central, sugerida pelo integrante Lucas, é unir a mecânica clássica de jogos de luta arcade (estilo *Street Fighter II*) com a identidade cultural de **Indaiatuba - SP**.

O jogo substitui os cenários genéricos por locais icônicos da nossa cidade, trazendo figuras inspiradas na vida real para o ringue digital. É um tributo à nostalgia dos anos 90 com um tempero local.
